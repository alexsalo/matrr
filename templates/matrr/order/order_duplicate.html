Please, read the following carefully!
To expedite the request process for return users MATRR allows you to create a duplicate request from a previously accepted request. This duplicate request cannot be modified, but it can be created to request tissues from another cohort, pending tissue availability. A rejected request cannot be duplicated. However, we can create a copy of an accepted/partially accepted request for you to resubmit. This copied order will use a special status "Duplicated" and will not interfere with your cart. Duplicated orders have specific restrictions, discussed below.

With Duplicated orders:
  • you can select a new cohort from which to request tissues
  • you can change details of the request
  • you can make changes to the requested tissues (monkeys, amount, units, notes etc.)
  • you can delete requested tissues
  • you cannot add new tissues. To request new tissues you must submit a new request.
  • you cannot select a cohort unless it all tissue originally requested are available in the new cohort
Once you have finished making changes, you have to resubmit the order. A duplicated order is just your draft. The MATRR team cannot see or process the order until it has been submitted.
