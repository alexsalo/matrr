MATRR Data Upload
The MATRR Data Symposium slides detailing what data to upload and how to format/upload data to MATRR can be found here.

In summary
DO
  • Format data consistently.
  • Add as many columns as you could possibly want. Instead of flagging or coloring data just add a new column
  • Name/label data consistently
    ('22hr' != '22 hour')
  • Describe your data and SOP, in detail, in a separate file
  • Use a standardized date format
  • Send us ALL your final data
DON'T
  • Rearrange columns
  • Insert new columns between existing columns. Append new columns to the end
  • Color/flag/mark data cells. Add new columns to denote/describe the data.
  • Merge cells together. All rows should have the same number of columns, all columns should have the same number of rows.
  • Merge disparate spreadsheets together
  • Send us images or graphs
  • Send us unfinished data
