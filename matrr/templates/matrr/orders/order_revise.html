Revise Request
For instructions on how to revise previous requests, see our FAQ page.
Please, read the following carefully!
A rejected request cannot be modified. However, we can create a copy of a rejected/partially rejected request for you to edit and resubmit. This copied request will use a special status "Revised" and will not interfere with your cart. Revised requests have specific restrictions, discussed below.

With Revised requests:
  • you can change details of the request
  • you can make changes to the requested tissues (monkeys, amount, units, notes etc.)
  • you can delete requested tissues
  • you cannot add new tissues. To request new tissues you must submit a new request.
  • you cannot change the cohort.
  • you cannot select monkey tissues that were accepted in partially rejected orders.
Once you have finished making changes, you have to resubmit the request. A revised request is just your draft. The MATRR team cannot see or process the request until it has been submitted.
