{% block title %}Duplicate Request{% endblock %}
Please, read the following carefully!
To expedite the request process for return users MATRR allows you to create a duplicate request from a previously submitted request. This copied request will use a special status "Duplicated" and will not interfere with your cart. Duplicated requests have specific restrictions, discussed below.

With Duplicated requests:
  • you can select a new cohort from which to request the same tissues
  • you can change details of the request
  • you can make changes to the requested tissues (monkeys, amount, units, notes etc.)
  • you can delete requested tissues
  • you cannot add new tissues. To request new tissues you must submit a new request.
Once you have finished making changes, you have to resubmit the request. A duplicated request is just your draft. The MATRR team cannot see or process the request until it has been submitted.
