MATRR Browser Support Requirements and Information

Problem

Its inconsistent compatibility with web standard prevents Internet Explorer from functioning correctly with this website.

Solutions

There are two known solutions to this issue.

  1. First, download a supported browser. Chrome and Firefox are widely used and both are free.
    Google Chrome is available here.
    Mozilla Firefox is available here.
  2. Alternatively, Google provides a addon for Internet Explorer which allows Internet Explorer to use Chrome's rendering engine on supported sites. This has been tested working on MATRR and is also free.
    ChromeFrame is available here.

    NOTE: When using chromeframe you must access the MATRR website thru our university-based domain for ChromeFrame to function properly.
